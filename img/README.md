# Portraits

Add client portraits to this folder as JPG or PNG files. The filename must match the `portrait` value in `js/data.js`.

Expected starter files:

- `gof.jpg`
- `zoltan-kodor.jpg`
- `hubert-mak.jpg`
- `antonio-costea.jpg`

Portraits work best in a vertical 4:5 crop with the face in the upper centre.
