# Portraits

Add client portraits to this folder as WebP, JPG or PNG files. The filename, extension and letter case must match the `portrait` value in `js/data.js`.

Expected starter files:

- `Gof.webp`
- `Zoltan.webp`
- `Hubert.webp`
- `Antonio Costea.webp`

Portraits work best in a vertical 4:5 crop with the face in the upper centre.
